# 📊 Portfolio Analyzer

A full-stack investment portfolio analyzer built with React + TypeScript and Node.js.

## Features
- Track stocks, ETFs, and crypto in one dashboard
- Real-time prices via Yahoo Finance & CoinGecko APIs
- AI-powered portfolio analysis with Claude
- P&L tracking, allocation charts, performance history
- Risk metrics: Sharpe ratio, volatility, diversification score

## Tech Stack
| Layer | Tech |
|-------|------|
| Frontend | React 18 + TypeScript + Vite |
| Styling | Tailwind CSS |
| State | Zustand |
| Charts | Recharts |
| Backend | Node.js + Express |
| AI | Claude API (Anthropic) |
| Prices | Yahoo Finance + CoinGecko |
| Deploy | Vercel + Railway |

## Getting Started

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
cp .env.example .env   # Add your API key
npm run dev
```
